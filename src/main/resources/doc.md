# API для взаимодействия SD и сайта технической поддержки

## Основные константы

1. Используемый формат данных - JSON
2. Авторизация реализуется при помощи передачи в токена в параметрах запроса в url
3. Используемый паттерн для даты - "dd.MM.yyyy", для даты/времени - "dd.MM.yyyy HH:mm:ss". Используется и в запросах и в ответах.
4. Во всех запросах фигурируют одни и те же параметры в url:

| Ключ      | Наименование         | Описание                                                                                     |
|-----------|----------------------|----------------------------------------------------------------------------------------------|
| func      | Функция              | Ссылка на метод внутри системы, пример "modules.partnerSupportApi.createSupportRequest"      |
| params    | Параметры            | Обычно передается фиксированное значение "request,response,user", иначе уточняется в методе. |
| raw       | Технический параметр | Значение всегда "true". Необходим из за особенностей платформы.                              |
| accessKey | Токен авторизации    | Выжается админом NSD.                                                                        |

5. В body ответа все результаты всегда обернуты в стандартный объект, имеющий два поля:

| Ключ   | Наименование | Описание                                                                                            |
|--------|--------------|-----------------------------------------------------------------------------------------------------|
| result | Результат    | Тип данных в данном поле не норматирован. Может быть null если при запросе была ошибка.             |
| error  | Ошибка       | Либо null, если ошибки нету, либо объект с полями code и message, если при запросе произошла ошибка |

6. Код ответа возвращается советующий результату выполнения:

- 400-499 при ошибке запроса
- 500 при ошибке сервера
- 200-299 при успешном запросе

## Методы

### modules.partnerSupportApi.createSupportRequest
#### Описание:
Создает обращение по услуге "Сервисное обслуживание" по переданным параметрам.
#### Метод запроса: POST
#### Описание данных запроса:
##### body:
Все передаваемые данные в body обязательны, кроме ключей в legalEntity, они обязательны при определенных условиях.

| Ключ                       | Наименование                            | Тип данных   | Обязательный                                            | Описание (если требуется)                                                                                                                                        |                                                                                                                                   
|----------------------------|-----------------------------------------|--------------|---------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| mode                       | Режим                                   | sting        | Да                                                      | Режим создания запроса. Поддерживает два значения: juristic и private, соотвесттвиющий созданию запросов от имени юридического и физического лица соотвествтенно |
| urgency                    | Срочность                               | object       | Да                                                      | Справочник получается отдельным запросом.                                                                                                                        |
| urgency.UUID               | UUID срочности                          | sting        | Да                                                      | UUID элемента справочника "срочность".                                                                                                                           |
| legalEntity                | Юридическое лицо                        | object       | Обязательный, если mode = juristic                      | Объект, содержащий информацию о юридическом лице.                                                                                                                |
| legalEntity.name           | Наименование юридического лица          | string       | Обязательный, если mode = juristic                      |                                                                                                                                                                  |
| legalEntity.inn            | ИНН юридического лица                   | number       | Обязательный, если mode = juristic                      |                                                                                                                                                                  |
| contactPerson              | Контактное лицо                         | object       | Да                                                      | Объект, содержащий информацию о юридическом лице / физическоим лице.                                                                                             |
| contactPerson.name         | ФИО котактного лица                     | string       | Да                                                      |                                                                                                                                                                  |
| contactPerson.email        | Email контактного лица                  | string       | Да                                                      |                                                                                                                                                                  |
| contactPerson.phone        | Телефон контактного лица                | string       | Да                                                      |                                                                                                                                                                  |
| supportObject              | Объект поддержки                        | object       | Да                                                      | Объект, содержащий информацию о объекте поддержки.                                                                                                               |
| supportObject.serialNumber | Серийный номер объекта поддержки        | string       | Да                                                      |                                                                                                                                                                  |
| supportObject.type         | Вид оборудования                        | object       | Да                                                      | Справочник получается отдельным запросом.                                                                                                                        |
| supportObject.type.UUID    | UUID вида оборудования                  | string       | Нет                                                     | UUID элемента справочника "вид оборудования".                                                                                                                    |
| supportObject.type.title   | Название вида оборудования              | string       | Обязательный, если не передан supportObject.type.UUID   | Название вида оборудования. Указывается, если отсутстует UUID (вид оборудования не из справочника).                                                              |
| supportObject.model        | Модель                                  | string       | Да                                                      |                                                                                                                                                                  |
| supportObject.address      | Адрес местонахождения объекта поддержки | string       | Да                                                      |                                                                                                                                                                  |
| supportObject.vendor       | Вендор                                  | object       | Да                                                      | Справочник получается отдельным запросом.                                                                                                                        |
| supportObject.vendor.UUID  | UUID вендор                             | string       | Нет                                                     | UUID элемента справочник "вендор".                                                                                                                               |
| supportObject.vendor.title | Название вендора                        | string       | Обязательный, если не передан supportObject.vendor.UUID | Название вендора. Указывается, если отсутстует UUID (вендор не из справочника).                                                                                  |

#### Описание данных ответа:
##### body:

| Ключ          | Наименование                  | Тип данных | Описание (если требуется) |
|---------------|-------------------------------|------------|---------------------------|
| result        | Созданное обращение           | object     |                           |
| result.UUID   | UUID созданно обращения       | string     |                           |
| result.number | Номер созданного обращения    | string     |                           |
| result.title  | Название созданного обращения | string     |                           |

### modules.partnerSupportApi.checkSupportObject
#### Описание:
Проверяет на существование объект поддержки по его идентификатору. Возврает информацию о нем, если найден. 404 статус если не найден.
#### Метод запроса: POST
#### Описание данных запроса:
##### body:

| Ключ          | Наименование   | Тип данных | Описание (если требуется)      |
|---------------|----------------|------------|--------------------------------|
| serialNumber  | Серийный номер | sting      | Используется для идентификации |

#### Описание данных ответа:
##### body:

| Ключ                         | Наименование                                                             | Тип данных | Описание (если требуется) |
|------------------------------|--------------------------------------------------------------------------|------------|---------------------------|
| result                       | Найденный объект поддержки                                               | object     |                           |
| result.title                 | Название объекта поддержки                                               | string     |                           |
| result.serialNumber          | Серийный номер объекта поддержки                                         | string     |                           |
| result.vendor                | Вендор                                                                   | object     |                           |
| result.vendor.title          | Название вендора                                                         | string     |                           |
| result.vendor.UUID           | UUID вендора                                                             | string     |                           |
| result.type                  | Вид оборудования                                                         | object     |                           |
| result.type.UUID             | UUID вида оборудования                                                   | string     |                           |
| result.type.title            | Название вида оборудования                                               | string     |                           |
| result.supportLevels         | Перечень условий обслуживания                                            | list       |                           |
| result.supportLevels[].title | Нзвание уровня обслуживания                                              | string     |                           |
| result.supportLevels[].UUID  | UUID уровня поддержки                                                    | string     |                           |
| result.contracts             | Перечень активных или ожидающих договоров/сертификатов объекта поддержки | list       |                           |
| result.contracts[].title     | Наименование договора/сертификата                                        | string     |                           |
| result.contracts[].UUID      | UUID договора/сертификата                                                | string     |                           |
| result.contracts[].dateStart | Дата начала действия включительно                                        | string     |                           |
| result.contracts[].dateEnd   | Дата окончания действия включительно                                     | string     |                           |

### Пример запроса:

```curl
curl --location 'https://pss.dev.ocs.ru/sd/services/rest/exec-post?func=modules.partnerSupportApi.checkSupportObject&params=request%2Cresponse%2Cuser&raw=true&accessKey=6e9a771b-c414-40df-acae-21cbbf28f9f9' \
--header 'Content-Type: application/json' \
--data '{
    "serialNumber": "1231231231231"
}'
```
